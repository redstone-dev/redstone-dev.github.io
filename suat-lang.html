suat :: a not-so-general-purpose scripting language

The name comes from SUper AuTomation.

Learn by example

Below is all you need to know about suat in a single example file.

You can run it using the suat compiler and it will compile successfully and run in most std-suat-compliant environments.


; this is a comment.
; comments need to be on their own line.
; also, no multiline comments :(

&require $NULL

; variables

var foo = "bar"
&echo($foo)

; 'var' to declare
; '$' to reference

@greet [name] ->

; this is a procedure.
&echo 

end
